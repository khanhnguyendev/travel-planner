# UI and UX Direction

## Visual Theme
Use a modern travel scrapbook aesthetic with clean cards, soft shadows, rounded corners, large place imagery, and a warm but premium palette.

## Tone
The product should feel collaborative, fun, and polished rather than spreadsheet-like.

## Design Keywords
- vibrant
- editorial
- soft depth
- social planning
- premium but playful

## Color Direction
- Sand or off-white background
- Ocean teal primary accent
- Coral secondary accent
- Muted category badges

## Core Screens
1. Marketing landing page
2. Dashboard with active projects
3. Project detail page with category boards
4. Place detail drawer or modal
5. Voting view
6. Expense tracker with receipt previews
7. Members and invite management
